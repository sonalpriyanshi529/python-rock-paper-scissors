# ✊ Rock Paper Scissors

A GUI-based Rock Paper Scissors game built with Python and Tkinter.

## 📸 Screenshot

![Game](screenshot.png)

## ✨ Features

- Play against the computer
- Random computer moves
- Score tracking
- Winner announcement
- GUI built using Tkinter

## 🛠️ Built With

- Python
- Tkinter

## 🚀 Run

```bash
python rock_paper_scissors.py
